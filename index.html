    在Linux上用两步安装TIM

    lampze 2018-12-26 13:40

    两行命令

    wget http://yun.tzmm.com.cn/index.php/s/5hJNzt2VR9aIEF2/download TIM.AppImage
chmod +x TIM.AppImage && ./TIM.AppImage

    介绍

    项目地址：Wine-QQ-TIM
    作者把 QQ 和 TIM 用AppImageKit打包了，现在只需要下载打好的包，就能体验完整的 QQ 和 TIM 了。
    跟以前自己安装 wine 比起来，真是省心太多了。

    问题解决

    列出一些我安装时发生的问题，我的系统为 gentoo

    • 'xterm-256color': unknown terminal type.
      TERM=xterm ./TIM.AppImage

      把上面的命令写在一个sh脚本里更方便

    • 系统重启后，TIM的数据消失

      因为作者把放数据的文件夹放到了 /tmp 下，有些系统会自动清理 /tmp 。
      请自行上网搜索自己发行版怎么关掉 /tmp 的自动清理。
      在此给出 gentoo 的解决办法。

      #打开/etc/init.d/bootmisc
#找到并注释下面这些代码

# Clean up /tmp directories
local tmp=
for tmp in ${clean_tmp_dirs:-${wipe_tmp_dirs-/tmp}}; do
   mountinfo -q -f tmpfs "$tmp" || cleanup_tmp_dir "$tmp"
done

    gentoo上安装最新版的pandoc

    lampze 2018-11-09 20:40

    起因

    emacs 的 org-mode 需要 pandoc 才能导出成html格式。而且 pandoc 版本太低，开启 emacs 时会报错，这就不能忍了。

    安装

    • 由于官方的 pandoc 版本太低，所以我们要加入 haskell overlay
      layman -a haskell
    • (可选) 开启 haskell overlay 的自动更新

      打开 /etc/portage/repos.conf/layman.conf 修改 haskell 字段为

      [haskell]
priority = 50
location = /var/lib/layman/haskell
layman-type = git
auto-sync = Yes
sync-type = git
sync-uri = https://github.com/gentoo-haskell/gentoo-haskell.git
    • 打开 /etc/portage/package.accept_keywords (如果是目录，就在这个目录下新建一个文件，名字无所谓) 加上
      */*::haskell ~amd64 #开启 haskell overlay 全部软件包的测试版
    • 接下来只需要安装pandoc就行了
      emerge -av pandoc

    解决NetworkManager不能开机自连wifi

    lampze 2018-11-01 09:11

    通用解决方法

    用root用户打开 /etc/NetworkManager/system-connections/ 目录下的wifi名称文件。
    找到 permissions 所在行，删除 = 后面的所有内容，保存退出就行了。

    plasma-nm用户

    在wifi设置里的 常规配置 下，选中 所有用户可以连接到此网络 就行了。

    原因猜想

    应该是因为wifi设置为只能指定用户连接，所以开机时没有自动连接。
