    GitHub 用户状态

    lampze 2019-01-11 20:17

    瞎BB

    经常逛 GitHub 的同学应该都注意到可以更改自己当前的状态了吧，这个小功能我觉得还是很不错的。以前看到自己关注的国外开发者突然几个星期没有绿点了，难免会觉得他是不是出事了。为什么感觉 GitHub 越来越向交友软件靠近了呢？
    话说自从微软收购 GitHub 后做了不少修改呀，私有仓库还免费了。
